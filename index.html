Front-End Stack

How I do things at Front-End?


I implement responsive design using Flexbox & Grids whether it is Mobile-first approach or Desktop-First Approach. I used Event Listeners & DOM Manipulation for User Interaction & Fetching Data .In terms of large scale design, I implement Sass for rapid, re-usable & more manageable styles. In terms of building SPA and large-scale projects I used React.js with Express.js or CSS Frameworks with PHP.

For fast pace front-end designs I use Material UI, mdbootstrap, Tailwind CSS, jQuery and more depending on the clients needs and project requirements. I always minified my stylesheets and scripts and compressed pictures to boost the app performance.

Back-End Stack

How I do things at Back-End?


I use PHP and Express.js in building data-driven web apps & RESTful APIs and continuously implement best practices in writing and securing web applications. I follow the Top 10 OWASP practices in protecting web apps from different attacks such as XSS, CSRF, SQL Injection, Directory Traversal, Brute Force and more. .

I implement normalization, aggregation, indexing, geospatial data and more intermediate techniques in designing RDMS using SQL with PHP or Mongoose with Express.js. I also use Git Version Control (branching, rebasing, pushing & pulling) & do README documentation. I don't yet have any experience in collaboration with Git however I abide with the best practices in the course Git for Teams by Kevin Bowersox. I am open to work /collaborate with teams.

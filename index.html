Back-End Stack

How I do things at Back-End?


I use PHP and Express in building data-driven web apps & RESTful APIs and continuously implement best practices in writing and securing web applications. I follow the Top 10 OWASP practices in protecting web apps from different attacks such as XSS, CSRF, SQL Injection, Directory Traversal, Brute Force and more. .

I implement normalization, aggregation, indexing, geospatial data and more intermediate techniques in designing RDMS using SQL with PHP/Laravel or Mongoose with Express.js. I also use Git Version Control (branching, rebasing, pushing & pulling) & do README documentation. I don't yet have any experience in collaboration with Git however I abide with the best practices in the course Git for Teams by Kevin Bowersox. I am open to work /collaborate with teams.
